机器人命令~

发表于 2022-11-03 更新于 2022-11-03
作者 Xiluo
2 分钟阅读

此处为一些常用功能 部分指令可能没有覆盖到

本篇博客图片较多 流量党情注意流量消耗

网络查询 (限制权限)

- /tracert <目标名称>	# 路由追踪
- /ping <目标名称> [ping次数] [超时时间(毫秒)]	# Ping连接测试
- /dns <记录名称> [记录类型] [自定义DNS]	# dns查询
- /web <URL链接> [编码]	# 网页测试
- /nmap <目标名称>	# 端口扫描
- /nmap <目标名称> <起始端口> <结束端口>	# 端口扫描
- /nmap <目标名称> <端口>	# 端口扫描
- /doh <域名> [类型]	# HTTPS加密DNS查询
- /gc <目标名称>	# 查墙
  1. 音乐
    • /music 点歌 <name> # 点首歌听吧~
    • /music 转换 <name> # 转换音乐分享
  2. 图片获取
    • /acg # 动漫图片!
    • /httpcat <code> # Funny Cat Pictures
  3. 方块人 (由 McMotd 插件提供)
    • /mcp <target> # 获取指定MC服务器的信息
    • /mcp # 获取指定MC服务器的信息
    • /mcadd <name> <address> # 为当前群聊绑定MC服务器
    • /mcdel <name> # 删除当前群聊绑定的服务器
  4. 娱乐
    1. 由 MuteGames 插件提供
      • (/)banMe # 自裁指令
      • (/)banMe <seconds> # 自裁指令
      • (/)duel # 决斗指令
      • (/)roulette # 俄罗斯轮盘指令
  5. 图片生成
    1. 由 PatPat 插件提供
      • 摸 + @目标 生成摸头Gif
    2. 由 DrawMeme 插件提供
      • #ph: Pxxnhub生成器
        • #ph + 文字 (会自动截取一半)
        • #ph + 文字 + 文字
      • #bw: 灰白照生成器 #bw + 文字 + 图片（若没有图片则要求获取）
      • #5000兆#5k兆
        • #5k兆 + 文字 (分词效果和Pxxnhub生成器一致)
        • #5k兆 + 文字 + 文字
      • emoji 合成
      • 发送任意两个emoji（不支持的emoji不会发送）
      • 0%生成器 使用#0即可
      • osu生成 #osu + 文字
      • marble大理石滤镜 #marble + 图片
      • flash假闪照 #flash + 图片
      • erode图像腐蚀
        • #erode + 图片

        • #erode + 数字 + 数字 + 数字 + 图片
